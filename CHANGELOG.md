# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added

- Initial release of Araviel AI Chat Assistant
- Multi-model support (Auto, Claude, ChatGPT, Gemini, Perplexity)
- Project-based conversation organization
- Dark and light theme support with system preference detection
- File attachment support for images and documents
- Web search toggle for enhanced responses
- Responsive design for mobile and desktop
- Keyboard shortcuts for common actions
- State persistence with redux-persist
- Toast notifications for user feedback
- Modal system for user interactions

### Components

- Chat interface with message streaming simulation
- Sidebar with projects and recent chats
- Welcome screen with suggestion cards
- Model selector dropdown
- Profile section with theme switcher
- Reusable UI components (Button, Input, Modal, etc.)

### Technical

- Next.js 14 with App Router
- TypeScript with strict mode
- Tailwind CSS with custom design system
- Redux Toolkit for state management
- Custom hooks for common patterns
- Comprehensive documentation
